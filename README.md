# ArknightsSurvivor_方舟幸存者

# 1 游戏介绍

**方舟幸存者**是本人在南京大学智能软件与工程学院2024秋C++高级程序设计课程中完成的期末项目，项目本身完成度并不算高。在项目中，我实现了一款类幸存者型游戏的无限模式，在一张由方舟基本地板组成的地图上，由玩家操作的角色和围绕在角色身边的子弹来击败地图边缘随机生成的敌人，随着时间推移，生成的怪物数量会显著提升，以此来提升游戏难度，而角色可以通过击败敌人释放技能强化自身。

# 2 角色介绍

# 2.1 维什戴尔

**维什戴尔：**是可供玩家选择的两名自机角色之一，初始时，维什戴尔身边会围绕三个由红色小球表示的**死魂灵**子弹，在技力条满后，维什戴尔可以使用技能**爆裂黎明**，~~技能释放后，立刻在攻击范围内召唤2个魂灵之影（最多存在3个，技能结束后保留），攻击力+180%，攻击间隔大幅增大，攻击时攻击力提升至220%，溅射范围大幅扩大且第一天赋的发动概率提高至100%，攻击装有6发弹药，打完后结束，可随时停止技能~~身边增加一个**死魂灵**子弹。

# 2.2 玛恩那

**玛恩那：**是可供玩家选择的两名自机角色之一，初始时，玛恩那身边会围绕三个由蓝色小球表示的**玛宝男**子弹，在技力条满后，玛恩那可以使用技能**未照耀的荣光**，技能释放后，~~攻击范围扩大，特性提升至2倍（每击倒一名敌人时特性倍率-10%），攻击对5个目标造成相当于180%攻击力的物理伤害。范围内所有敌人受到卡西米尔干员攻击时额外附带玛恩纳12%攻击力的真实伤害~~立即向四周释放四道**未照耀的荣光**子弹。

# 3 敌方介绍

# 3.1 源石虫

**源石虫：**野生的被感染生物。这种低智能生物极有可能会被敌方术师操纵，以进入人工设施内发动集群攻击。血量较低，移速较快

# 3.2 南京野猪

**南京野猪：**这种恐怖的生物广泛见于南京大学仙林校区，具有巨大的体型和超高的移速，但似乎是吃撑了的样子，此时此刻移动比较缓慢。血量较高，移速较低
